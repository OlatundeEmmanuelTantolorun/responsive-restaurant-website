# Responsive Restaurant Website

A fully responsive multi-page restaurant website built using HTML, CSS, and Bootstrap 5.

## Features

- Responsive design
- 4 Html pages (Home, About, Menu, Contact)
- Bootstrap components
- Clean layout structure

## Tech Stack

- HTML5
- CSS#
- Bootstrap 5

## Live Demo

The website is deployed on vercel.
Web analytics are enabled to track visitor traffic in real time.
